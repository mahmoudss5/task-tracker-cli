# Task Tracker CLI

A simple command-line interface (CLI) to track and manage your tasks.
Built with Java (No external libraries used).

## Features
- Add, Update, and Delete tasks.
- Mark tasks as In Progress or Done.
- List all tasks or filter by status (done, todo, in-progress).
- Data is persisted in a JSON file.
